Bootstrapped personal page

A twitter Bootstrap based personal page theme.

Geared toward graduate students and faculty of the Computer Science Department at Ben-Gurion University of the Negev, this no-fuss, single file theme should be easy to install and edit with only minimal knowledge of HTML.

To get started:

  1. Download the theme from here: boot.zip.
  2. Go to your Linux home directory (~username).
  3. Create a folder names ".html" (mkdir .html).
  4. Extract the contents of boot.zip to ~/.html/.
  5. Set permission for your home directory and .html directory:
    chmod 711 ~username
    chmod 711 ~/.html/
    chmode -R 755 ~/.html/*
Be sure to replace "username" with your Linux user. Also - keep in mind that the contense of your .html directory is publicly visible so avoid storing anything embarrassing there...

If you are not a member of the BGU CS dept. you are still welcome to use this theme - extracting the contents of boot.zip to your site directory of any web-server should work just fine.

Once you have completed the above steps - edit index.html in your .html directory to customize your site. Don't forget to change "Your Name" to your actual name - both in the navigation menu and the page title!

Navigation tabs can be renamed/removed by editing the #navbar section of the file.

If you are completely unfamiliar with HTML, you might want to read a guide first.
